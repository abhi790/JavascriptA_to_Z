We will going to learn LocalStorage in this lesson

LocalStorage has 4 function, so that we can use it to store, access and remove content from and to localstorage

  • localstorage.setItem('key', 'value')
  • localstorage.getItem('key')
  • localstorage.removeItem('key')
  • localstorage.clear()

1. Storing single string value into localstorage

        // 1. Storing single string
        localStorage.setItem("fruit", "Mango");
        // accessing single value using it's key
        console.log(localStorage.getItem("fruit"));

        let token = "Cartoon";
        let tokenValue = "Oggy and Cockroaches";
        localStorage.setItem(token, tokenValue);

        //Both will result same output
        console.log(localStorage.getItem(token));
        console.log(localStorage.getItem("Cartoon"));

        // clear a single item from localstorage
        localStorage.removeItem("Cartoon");

        //clear complete local storage
            localStorage.clear();

2. Storing key as string and value by converting JSON.stringify as string 

        // 2. Storing key as string and value as JSON.stringify as string
        let product = {
            name: "abhimanyu",
            age: 28,
            address: {
            pincode: 700107,
            city: "kolkata",
            state: "West Bengal",
            },
        };

        //    // storing object in localstorage
        //    // first convert the object into stringify json
        let stringifyJson = JSON.stringify(product);

        // don't use only product as key instead use "product" as string
        localStorage.setItem("product", stringifyJson);

        //Accessing the object stored in localstorage by its key
